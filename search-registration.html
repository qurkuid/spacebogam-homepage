공간보감 검색엔진 등록 안내

이 페이지는 공개 색인용이 아니라, 네이버·구글 등록 때 사장님이 인증만 진행할 수 있도록 정리한 내부 안내입니다.

현재 배포 URL
https://spacebogam.kr/

도메인 후보
spacebogam.kr — 구매/연결 후 canonical, robots, sitemap 기준 URL을 교체합니다.

1. 도메인 연결

도메인 구매 후 DNS에서 아래 중 하나로 연결합니다.

  • GitHub Pages custom domain: CNAME 또는 apex용 A record
  • 후보 도메인: spacebogam.kr
  • 도메인이 확정되면 repo 루트에 CNAME 파일을 추가합니다.

2. 네이버 서치어드바이저

  1. 네이버 서치어드바이저 접속
  2. 사이트 등록: 현재 URL 또는 확정 도메인 입력
  3. 소유확인 방식 선택: HTML 파일 업로드 또는 meta 태그
  4. 인증 파일명/메타값을 저에게 보내면 파일에 반영
  5. 인증 버튼 클릭
  6. sitemap.xml, feed.xml, robots.txt 제출

3. Google Search Console

  1. Google Search Console 접속
  2. URL-prefix 또는 Domain 속성 등록
  3. HTML 파일 또는 DNS TXT 인증값 선택
  4. 인증값을 저에게 보내면 파일/DNS 안내에 반영
  5. sitemap.xml 제출

4. 제출할 파일

  • /robots.txt
  • /sitemap.xml
  • /feed.xml
  • /blog.html
  • /portfolio.html
