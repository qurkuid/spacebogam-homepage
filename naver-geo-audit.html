Naver GEO Audit

공간보감 네이버 검색 최적화 치명점

현재 GitHub Pages 배포본은 기술 SEO는 보강됐지만, 네이버 지역검색에서는 엔티티 신뢰가 가장 큰 병목입니다.

가장 심각한 문제

네이버 플레이스, 네이버 블로그, 실제 주소, 전화번호, 사업자 정보가 페이지와 강하게 연결되어 있지 않습니다. 네이버는 지역 업체 판단에서 이 신호가 매우 중요합니다.

GitHub Pages 도메인도 임시 사이트처럼 보일 수 있어, 실제 운영 도메인 또는 intm.kr 하위 경로 연결이 필요합니다.

즉시 보강한 것

Yeti robots 허용, 지역/서비스 랜딩 페이지 6개, sitemap 확장, 자연스러운 GEO 문구, 상담 신청 연결을 반영했습니다.
